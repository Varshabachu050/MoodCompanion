# 🧠 Mood Health Game App

A **Flask-based interactive mental wellness web application** that helps users improve their mood through small relaxing and engaging games.

Users select their **current mood**, and the application provides a **short game designed to improve that feeling**.
At the end of each game, the system returns a **motivational quote based on the user’s mood and performance**.

This project combines **mental wellness concepts, simple browser games, and web development** into a single application.

---

# 🌟 Features

* 🎭 Mood selection interface
* 🎮 Different mini-games for different emotions
* ⏱ Score and timer based gameplay
* 💬 Motivational quotes after completing a game
* 🎨 Interactive browser-based games
* 🌈 Clean UI with relaxing color themes

---

# 🎮 Mood Based Games

| Mood      | Game                             |
| --------- | -------------------------------- |
| 😊 Happy  | Balloon Pop Game                 |
| 😢 Sad    | Math Bubbles (solve simple math) |
| 💖 Lonely | Catch the Kind Hearts            |
| 😴 Tired  | Relax & Paint creativity game    |
| 😡 Angry  | Smash the Anger Blocks           |
| 🧘 Calm   | Memory Flip Game                 |

Each game includes:

* Score tracking
* Timer system
* Interactive gameplay
* Motivational quote after completion

# ▶️ How to Run the Project

### 1️⃣ Clone the repository

```bash
git clone https://github.com/Varshabachu050/MoodChampnion.git
```

### 2️⃣ Navigate to the project folder

```bash
cd mood-health-app
```

### 3️⃣ Install Flask

```bash
pip install flask
```

### 4️⃣ Run the application

```bash
python app.py
```

### 5️⃣ Open in browser

```
http://127.0.0.1:5000
```
